# 🧮 Simple JS Calculator

A sleek, functional calculator web application built with vanilla JavaScript, HTML5, and CSS3. This project focuses on DOM manipulation, CSS Grid layouts, and clean event-handling logic.

---

## 🚀 Features
* **Standard Operations:** Addition, subtraction, multiplication, and division.
* **Grid Layout:** Responsive button interface using CSS Grid.
* **Error Handling:** Prevents crashes from invalid mathematical expressions.
* **Clear & Delete:** Functions to reset the display or remove the last digit entered.
* **Dark Theme:** High-contrast UI designed for readability.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for the calculator interface.
* **CSS3:** Custom styling using Flexbox and Grid.
* **JavaScript (ES6):** Logic for calculations and UI updates.
